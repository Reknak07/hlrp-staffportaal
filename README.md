# Haarlem Roleplay Staff Portal

Een volledig vernieuwde, premium staff portal voor Haarlem Roleplay FiveM server.

## 🚀 Kenmerken

### Design
- **Donkere premium UI** met glassmorphism effecten
- **Glowing borders** en gradient overlays
- **Floating background orbs** met animaties
- **Smooth transitions** en hover effects
- **Responsive design** voor alle apparaten

### Functionaliteiten
- **Dashboard** met statistieken, snelle acties en meldingen
- **Staffregels** met documentatie layout en zoekfunctie
- **Staffrangen** met expandable hiërarchie panels
- **Strafbeleid** met premium tabel en filters
- **Strafberekenaar** - werkende tool voor sancties
- **Tickets** overzicht met categorieën
- **Sollicitaties** informatie

### Extra Features
- **Command Palette** (Ctrl+K) voor snelle navigatie
- **Global search** in sidebar
- **Scroll progress bar**
- **Notification systeem**
- **Loading animations**
- **Mobile menu** voor responsive design

## 📋 Gebruik

Open simpelweg `index.html` in je browser.

Geen server of build process nodig - het is een volledig standalone project.

## 📱 Pagina's

1. **Dashboard** - Overzicht met statistieken en snelle acties
2. **Staffregels** - Complete staff regelgeving met zoekfunctie
3. **Staffrangen** - Volledige hiërarchie met alle rangen
4. **Strafbeleid** - Alle strafcategorieën in een premium tabel
5. **Strafberekenaar** - Interactieve tool voor sanctie berekening
6. **Tickets** - Overzicht van alle ticketcategorieën
7. **Sollicitaties** - Informatie over staff sollicitatie proces

## ⌨️ Keyboard Shortcuts
